Support

Accessing Ask AI

You can access your Ask AI account by visiting ask-ai.cabinetoffice.gov.uk. You’ll need to sign in then put in your email address and select the submit button.

Signing in

If you’re having trouble signing in to Ask AI, check that:

  • you’re using your work email address
  • you’ve typed your work email address correctly

You should check your junk or spam folder for an email from Ask AI.

Error messages

This text has been flagged as potentially containing sensitive data. Are you sure this is an appropriate query?

You need to review the prompt you have entered and make sure it is not sensitive before submitting. Sensitive data can be anything from a phone number through to a prompt that contains security information.

Bug or site vulnerability

If you’ve found an issue or a vulnerability, contact us using the details at the bottom of this page including as much information as possible.

Website not loading

If the website is not loading, please close the browser tab and try again later.

Contact us

If you’ve read the information above but haven’t been able to find a solution to your problem, please email us at ask-ai-support@cabinetoffice.gov.uk including as much information as possible.
